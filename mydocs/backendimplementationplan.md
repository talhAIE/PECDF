## 2. Database Selection & Justification

### Recommended: SQLite (Development) → PostgreSQL (Production)
**ORM:** SQLAlchemy 2.0 with Alembic for migrations

### Why SQLite for Now
| Criterion | SQLite | PostgreSQL | MongoDB |
|-----------|--------|------------|---------|
| Setup complexity | Zero (file-based) | Needs server install | Needs server install |
| Dev speed | Instant | Moderate | Moderate |
| FYP scale | Perfect | Overkill | Overkill |
| Migration to prod | One line change in `.env` | Already there | Full rewrite |
| Relational queries | Full SQL support | Full SQL support | No joins |
| JSON columns | Supported | Better | Native |

**Decision:** SQLite with SQLAlchemy ORM. When you deploy to production, change `DATABASE_URL` to `postgresql://...` — zero code changes needed because SQLAlchemy abstracts the driver.

### What Needs a Database vs What Stays In-Memory

| Data | Storage | Reason |
|------|---------|--------|
| ML model + master_df | In-memory (loaded at startup) | Speed — every request needs it |
| Agent sessions (active) | In-memory dict | Speed — sub-millisecond lookup |
| Agent conversation history | Database | Persist across restarts, retrieve by session ID |
| Generated forecasts | Database | Cache so same request doesn't re-run model |
| Generated reports | Database | User can re-download without regenerating |
| Scenario runs | Database | History for the AI agent to reference |
| Commodity metadata | Hardcoded in config | Never changes |

### Table 1 — `agent_sessions`
Represents one browser chat session. Created when user opens the AI Chat page.

```sql
CREATE TABLE agent_sessions (
    id            VARCHAR(36)  PRIMARY KEY,   -- UUID, generated by backend
    user_id       VARCHAR(36)  NOT NULL,       -- Supabase auth.users.id (sub claim)
    created_at    TIMESTAMP    NOT NULL DEFAULT CURRENT_TIMESTAMP,
    last_active   TIMESTAMP    NOT NULL DEFAULT CURRENT_TIMESTAMP,
    macro_pkr     REAL,                        -- macro context when session started
    macro_oil     REAL,
    macro_conf    REAL,
    message_count INTEGER      NOT NULL DEFAULT 0
);

CREATE INDEX idx_sessions_user ON agent_sessions(user_id);
```

### Table 2 — `agent_messages`
Every user and assistant message within a session.

```sql
CREATE TABLE agent_messages (
    id          INTEGER      PRIMARY KEY AUTOINCREMENT,
    session_id  VARCHAR(36)  NOT NULL REFERENCES agent_sessions(id) ON DELETE CASCADE,
    role        VARCHAR(10)  NOT NULL,   -- 'user' | 'assistant'
    content     TEXT         NOT NULL,
    tools_used  TEXT,                    -- JSON array: ["forecast_tool", "momentum_tool"]
    created_at  TIMESTAMP    NOT NULL DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_messages_session ON agent_messages(session_id);
CREATE INDEX idx_messages_created ON agent_messages(created_at);
```

### Table 3 — `forecasts`
Every forecast request. Used as a cache — same inputs return cached result.

```sql
CREATE TABLE forecasts (
    id             INTEGER      PRIMARY KEY AUTOINCREMENT,
    user_id        VARCHAR(36)  NOT NULL,       -- Supabase auth.users.id
    session_id     VARCHAR(36),                 -- nullable — can be called outside agent
    hs_code        VARCHAR(10)  NOT NULL,
    commodity_name VARCHAR(100) NOT NULL,
    start_month    INTEGER      NOT NULL,        -- YYYYMM format
    n_months       INTEGER      NOT NULL,
    usd_pkr        REAL         NOT NULL,
    brent_oil      REAL         NOT NULL,
    us_confidence  REAL         NOT NULL,
    created_at     TIMESTAMP    NOT NULL DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_forecasts_user ON forecasts(user_id);
CREATE INDEX idx_forecasts_hs ON forecasts(hs_code);
CREATE INDEX idx_forecasts_month ON forecasts(start_month);
```

### Table 4 — `forecast_results`
Individual monthly predictions within a forecast run. One row per month.

```sql
CREATE TABLE forecast_results (
    id            INTEGER   PRIMARY KEY AUTOINCREMENT,
    forecast_id   INTEGER   NOT NULL REFERENCES forecasts(id) ON DELETE CASCADE,
    month         INTEGER   NOT NULL,    -- YYYYMM
    predicted_usd REAL      NOT NULL,
    lower_bound   REAL      NOT NULL,
    upper_bound   REAL      NOT NULL,
    step_number   INTEGER   NOT NULL     -- 1=first month, 2=second, etc.
);

CREATE INDEX idx_results_forecast ON forecast_results(forecast_id);
```

### Table 5 — `scenarios`
Every scenario (what-if) run. One record per scenario configuration.

```sql
CREATE TABLE scenarios (
    id             INTEGER      PRIMARY KEY AUTOINCREMENT,
    user_id        VARCHAR(36)  NOT NULL,       -- Supabase auth.users.id
    session_id     VARCHAR(36),
    hs_code        VARCHAR(10)  NOT NULL,
    commodity_name VARCHAR(100) NOT NULL,
    variable_name  VARCHAR(20)  NOT NULL,  -- 'pkr' | 'oil' | 'conf'
    target_month   INTEGER      NOT NULL,
    n_months       INTEGER      NOT NULL DEFAULT 1,
    range_min      REAL         NOT NULL,
    range_max      REAL         NOT NULL,
    steps          INTEGER      NOT NULL DEFAULT 10,
    fixed_pkr      REAL,
    fixed_oil      REAL,
    fixed_conf     REAL,
    created_at     TIMESTAMP    NOT NULL DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_scenarios_user ON scenarios(user_id);
```

### Table 6 — `scenario_results`
One row per data point in the scenario chart.

```sql
CREATE TABLE scenario_results (
    id           INTEGER   PRIMARY KEY AUTOINCREMENT,
    scenario_id  INTEGER   NOT NULL REFERENCES scenarios(id) ON DELETE CASCADE,
    input_value  REAL      NOT NULL,   -- the varied variable value

```

### Table 7 — `reports`
AI-generated written export outlook reports.

```sql
CREATE TABLE reports (
    id            INTEGER      PRIMARY KEY AUTOINCREMENT,
    user_id       VARCHAR(36)  NOT NULL,       -- Supabase auth.users.id

    brent_oil     REAL         NOT NULL,
    us_confidence REAL         NOT NULL,

    created_at    TIMESTAMP    NOT NULL DEFAULT CURRENT_TIMESTAMP

CREATE INDEX idx_reports_user ON reports(user_id);
CREATE INDEX idx_reports_session ON reports(session_id);
CREATE INDEX idx_reports_created ON reports(created_at);
```

## 4. Supabase Authentication Design

### How It Works — Full Flow

```
┌─────────────┐      1. Sign up / Log in       ┌──────────────┐
│   Frontend  │ ─────────────────────────────→ │   Supabase   │
│  (React)    │ ←───────────────────────────── │    Auth      │
│             │      2. JWT access token        └──────────────┘
│             │
│             │      3. Every API request
│             │         Authorization: Bearer <JWT>
│             │ ─────────────────────────────→ ┌──────────────┐
│             │ ←───────────────────────────── │   FastAPI    │
│             │      4. Protected response      │   Backend    │
└─────────────┘                                └──────────────┘
                                                     │
                                          5. Verify JWT using
                                             SUPABASE_JWT_SECRET
                                          6. Extract user_id from token
                                          7. Scope all DB queries
                                             to that user_id
```

**Key principle:** Supabase handles ALL authentication logic (signup, login, password reset, OAuth). FastAPI only verifies the JWT token on each request — it never stores passwords or manages auth state.

### What Supabase Manages (you configure in Supabase dashboard)
- Email/password registration and login
- Google OAuth (optional, enable in Supabase dashboard → Auth → Providers)
- Password reset emails (automatic)
- JWT token generation and refresh
- User table (`auth.users`) — stored in Supabase, not your local DB

### What FastAPI Manages
- JWT verification on every protected endpoint
- Mapping `user_id` (from JWT) to local database rows
- Ensuring User A cannot see User B's forecasts/reports/sessions

### `middleware/auth.py` — JWT Verification

```python
from fastapi import HTTPException, Security, Depends
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
from jose import jwt, JWTError
from config import settings

security = HTTPBearer()

def get_current_user(
    credentials: HTTPAuthorizationCredentials = Security(security)
) -> dict:
    """
    FastAPI dependency — verifies Supabase JWT on every protected request.
    Returns the decoded token payload (contains user_id, email, etc.)

    Usage: add  current_user: dict = Depends(get_current_user)
           to any endpoint that requires authentication.
    """
    token = credentials.credentials
    try:
        payload = jwt.decode(
            token,
            settings.supabase_jwt_secret,
            algorithms=["HS256"],
            options={"verify_aud": False}   # Supabase doesn't set audience
        )
        user_id: str = payload.get("sub")
        if not user_id:
            raise HTTPException(status_code=401, detail="Invalid token: no user ID")
        return {"user_id": user_id, "email": payload.get("email")}

    except JWTError as e:
        raise HTTPException(status_code=401, detail=f"Token invalid or expired: {str(e)}")


# Convenience alias for use in routers
CurrentUser = Depends(get_current_user)
```

### `routers/auth.py` — Auth Endpoints

```python
from fastapi import APIRouter, Depends
from middleware.auth import get_current_user

router = APIRouter(prefix="/auth", tags=["Auth"])

@router.get("/me")
def get_me(current_user: dict = Depends(get_current_user)):
    """Returns the logged-in user's info. Use to verify token is valid."""
    return {
        "user_id": current_user["user_id"],
        "email":   current_user["email"]
    }

@router.get("/verify")
def verify_token(current_user: dict = Depends(get_current_user)):
    """Simple token check — 200 means valid, 401 means invalid/expired."""
    return {"valid": True, "user_id": current_user["user_id"]}
```

### Protecting Endpoints

Add `current_user: dict = Depends(get_current_user)` to any route that needs auth:

```python
# Public endpoint — no auth needed

# Protected endpoint — requires valid Supabase JWT
@router.post("/forecast/multi-horizon")

    req: MultiHorizonForecastRequest,
    db: Session = Depends(get_db),
    current_user: dict = Depends(get_current_user)   # ← auth guard

    user_id = current_user["user_id"]
    # All DB operations scoped to this user_id
    ...
```

### Public vs Protected Endpoints

| Endpoint | Auth Required | Reason |
|----------|--------------|---------|
| `GET /health` | No | System monitoring |
| `GET /commodities` | No | Public info |
| `POST /forecast/*` | **Yes** | User-specific forecasts saved to DB |
| `POST /scenario/*` | **Yes** | User-specific scenario history |
| `GET /seasonality/*` | No | Static analytical data |
| `GET /momentum` | No | Static analytical data |
| `GET /historical/*` | No | Static analytical data |
| `POST /agent/chat` | **Yes** | Per-user conversation sessions |
| `POST /agent/report` | **Yes** | Per-user saved reports |
| `GET /auth/me` | **Yes** | Requires valid token to return info |

### Module 2 — `database/`
**Build second. Defines storage before services need it.**

**`database/connection.py`**
```python
from sqlalchemy import create_engine
from sqlalchemy.orm import sessionmaker, DeclarativeBase
from config import settings

engine = create_engine(
    settings.database_url,
    connect_args={"check_same_thread": False}  # SQLite only
)
SessionLocal = sessionmaker(autocommit=False, autoflush=False, bind=engine)

class Base(DeclarativeBase):
    pass

def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()
```

4. Write `crud.py` — all read/write functions
5. Test: create a session, add a message, retrieve it

### Module 3 — `ml/loader.py`
**Build third. Services depend on the model being loaded.**

```python
# ml/loader.py
import joblib
import pandas as pd
from pathlib import Path
from config import settings

# Global state — loaded once at startup
model = None
FEATURE_COLS = None
hs_categories = None
HS_LABELS = None
train_cutoff = None
test_mape = None
test_r2 = None
master_df = None

def load_artifacts():
    global model, FEATURE_COLS, hs_categories, HS_LABELS
    global train_cutoff, test_mape, test_r2, master_df

    artifact = joblib.load(Path(settings.model_path))
    model         = artifact["model"]
    FEATURE_COLS  = artifact["feature_cols"]
    hs_categories = artifact["hs_categories"]
    HS_LABELS     = artifact["hs_labels"]
    train_cutoff  = artifact["train_cutoff"]
    test_mape     = artifact["test_mape"]
    test_r2       = artifact["test_r2"]

    master_df = pd.read_csv(
        Path(settings.master_data_path),
        dtype={"HS_Code": str}
    )

    print(f"Model loaded — MAPE: {test_mape}%, R²: {test_r2}")
    print(f"Master dataset: {master_df.shape[0]} rows")
```

Called in `app.py` lifespan handler so it runs exactly once on startup.

```python
def forecast_n_months(hs_code, start_yyyymm, n_months,
                      usd_pkr, brent_oil, us_conf):
    """
    Returns list of ForecastPoint dicts with confidence bands.
    Uses UNCERTAINTY_GROWTH to widen bands per step.
    """
    forecast_buffer = {}
    results = []
    start_dt = pd.to_datetime(str(start_yyyymm), format="%Y%m")

    for i in range(n_months):
        month = int((start_dt + pd.DateOffset(months=i)).strftime("%Y%m"))
        pred = make_prediction(hs_code, month, usd_pkr, brent_oil,
                               us_conf, forecast_buffer)
        forecast_buffer[(str(hs_code), month)] = pred

        base_uncert = loader.test_mape / 100
        factor = UNCERTAINTY_GROWTH[min(i, len(UNCERTAINTY_GROWTH)-1)]
        results.append({
            "month": month,
            "predicted_m": round(pred / 1e6, 3),
            "lower_bound": round(max(0, pred * (1 - base_uncert * factor)) / 1e6, 3),
            "upper_bound": round(pred * (1 + base_uncert * factor) / 1e6, 3),
            "step": i + 1
        })
    return results
```

```python
import numpy as np
from services.forecast_service import forecast_n_months

def run_single_variable_scenario(hs_code, target_yyyymm, n_months,
                                  variable, range_min, range_max, steps,
                                  fixed_pkr, fixed_oil, fixed_conf):
    """
    Vary one macro variable across a range.
    Returns list of {input_value, predicted_m} points.
    """
    values = np.linspace(range_min, range_max, steps)
    points = []

    for v in values:
        pkr  = v if variable == "pkr"  else fixed_pkr
        oil  = v if variable == "oil"  else fixed_oil
        conf = v if variable == "conf" else fixed_conf

        forecast = forecast_n_months(hs_code, target_yyyymm,
                                     n_months, pkr, oil, conf)
        avg_pred = sum(f["predicted_m"] for f in forecast) / len(forecast)
        points.append({"input_value": round(v, 2), "predicted_m": round(avg_pred, 3)})

    # Compute linear slope for annotation
    x = [p["input_value"] for p in points]
    y = [p["predicted_m"] for p in points]
    slope = np.polyfit(x, y, 1)[0]

    return points, slope

def run_multi_variable_scenario(hs_code, target_yyyymm, n_months,
                                 pkr_values, oil_values, fixed_conf):
    """
    Vary PKR × Oil simultaneously — returns a matrix.
    """
    matrix = {}
    for pkr in pkr_values:
        matrix[pkr] = {}
        for oil in oil_values:
            forecast = forecast_n_months(hs_code, target_yyyymm,
                                         n_months, pkr, oil, fixed_conf)
            avg = sum(f["predicted_m"] for f in forecast) / len(forecast)
            matrix[pkr][oil] = round(avg, 3)
    return matrix
```

```python
import pandas as pd
import numpy as np
from ml import loader

def get_seasonality(hs_code):
    """Historical average export per month (Jan–Dec)."""
    df = loader.master_df[loader.master_df["HS_Code"] == str(hs_code)].copy()
    monthly = df.groupby("Month")["Export_Value_USD"].mean()
    monthly_m = (monthly / 1e6).round(3)
    peak_month = int(monthly_m.idxmax())
    trough_month = int(monthly_m.idxmin())
    return {
        "monthly_averages": monthly_m.to_dict(),
        "peak_month": peak_month,
        "trough_month": trough_month,
        "seasonality_strength": round(
            (monthly_m.max() - monthly_m.min()) / monthly_m.mean() * 100, 1
        )
    }

def get_momentum(hs_code=None):
    """
    3M and 6M momentum for one or all commodities.
    Momentum = % change vs N months ago.
    """
    codes = [hs_code] if hs_code else loader.hs_categories
    results = []
    for hs in codes:
        df = (loader.master_df[loader.master_df["HS_Code"] == hs]
              .sort_values("Date_YYYYMM").tail(12))
        if len(df) < 6:
            continue
        last  = df["Export_Value_USD"].iloc[-1]
        m3ago = df["Export_Value_USD"].iloc[-4]
        m6ago = df["Export_Value_USD"].iloc[-7] if len(df) >= 7 else np.nan

        mom3 = round((last - m3ago) / m3ago * 100, 1) if m3ago != 0 else 0
        mom6 = round((last - m6ago) / m6ago * 100, 1) if (not np.isnan(m6ago) and m6ago != 0) else 0

        results.append({
            "hs_code": hs,
            "commodity": loader.HS_LABELS[hs],
            "momentum_3m_pct": mom3,
            "momentum_6m_pct": mom6,
            "direction": "up" if mom3 > 2 else ("down" if mom3 < -2 else "flat"),
            "last_actual_m": round(last / 1e6, 3)
        })
    return results

def get_currency_sensitivity(target_yyyymm, n_months, pkr_min,
                              pkr_max, fixed_oil, fixed_conf):
    """
    For all commodities, compute how much predicted exports change
    per 10 PKR movement.
    """
    from services.forecast_service import forecast_n_months
    pkr_values = [pkr_min, pkr_min+10, pkr_max-10, pkr_max]
    results = []

    for hs in loader.hs_categories:
        preds = []
        for pkr in pkr_values:
            fc = forecast_n_months(hs, target_yyyymm, n_months,
                                   pkr, fixed_oil, fixed_conf)
            preds.append(sum(f["predicted_m"] for f in fc) / len(fc))

        slope = np.polyfit(pkr_values, preds, 1)[0] * 10  # per 10 PKR
        results.append({
            "hs_code": hs,
            "commodity": loader.HS_LABELS[hs],
            "change_per_10pkr_m": round(slope, 3),
            "direction": "increases" if slope > 0 else "decreases"
        })
    return sorted(results, key=lambda x: abs(x["change_per_10pkr_m"]), reverse=True)

def get_historical(hs_code, months=24):
    """Last N months of actual export data for a commodity."""
    df = (loader.master_df[loader.master_df["HS_Code"] == str(hs_code)]
          .sort_values("Date_YYYYMM").tail(months))
    return [
        {"month": int(row["Date_YYYYMM"]),
         "export_value_m": round(row["Export_Value_USD"] / 1e6, 3)}
```

```python
from langchain_groq import ChatGroq
from langchain.agents import create_tool_calling_agent, AgentExecutor
from langchain.memory import ConversationBufferWindowMemory
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from agent.tools import ALL_TOOLS
from agent.prompts import build_system_prompt
from config import settings

# In-memory session store: {session_id: AgentExecutor}
_sessions: dict[str, AgentExecutor] = {}

def get_or_create_agent(session_id: str, macro_pkr: float,
                         macro_oil: float, macro_conf: float) -> AgentExecutor:
    if session_id in _sessions:
        return _sessions[session_id]

    # Groq — free, ultra-fast, Llama 3.3 70B
    llm = ChatGroq(
        model=settings.agent_model,
        api_key=settings.groq_api_key,
        max_tokens=settings.agent_max_tokens
    )

    prompt = ChatPromptTemplate.from_messages([
        ("system", build_system_prompt(macro_pkr, macro_oil, macro_conf)),
        MessagesPlaceholder("chat_history"),
        ("human", "{input}"),
        MessagesPlaceholder("agent_scratchpad"),
    ])

    memory = ConversationBufferWindowMemory(
        k=settings.agent_memory_window,
        memory_key="chat_history",
        return_messages=True
    )

    agent = create_tool_calling_agent(llm, ALL_TOOLS, prompt)
    executor = AgentExecutor(agent=agent, tools=ALL_TOOLS,
                             memory=memory, verbose=True)

    _sessions[session_id] = executor
    return executor

def clear_session(session_id: str):
    _sessions.pop(session_id, None)
```
